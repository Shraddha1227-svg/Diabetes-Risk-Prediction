# Diabetes-Risk-Prediction
This project applies Multiple Linear Regression, Stepwise Regression, and LASSO Regression to analyze the impact of health and lifestyle factors on diabetes risk. Using a dataset of 500 observations and 20 predictor variables,the study identifies the most significant factors influencing diabetes risk and compares different regression-based feature selection techniques.

Project Overview

The objective of this project is to build an accurate regression model for predicting diabetes risk and to identify the most influential health and lifestyle variables. The analysis includes model fitting, residual diagnostics, multicollinearity testing, subset selection, and LASSO regression to improve model performance and interpretability.

Key Findings
The regression model explains approximately 98% of the variation in diabetes risk (R² ≈ 0.98).
Family history of diabetes is the strongest predictor.
Age, BMI, screen time, and triglyceride levels increase diabetes risk.
Physical activity and a healthy diet significantly reduce diabetes risk.
Stepwise Regression produces a simpler and more interpretable model.
LASSO Regression provides better feature selection for predictive modeling.
